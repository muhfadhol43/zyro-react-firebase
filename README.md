# ZYRO Ecommerce

Modern ecommerce web application built with React, Tailwind CSS, and Firebase.

## Features

- Product listing
- Product detail page
- Shopping cart
- Quantity management
- Google Authentication
- Smooth animations
- Responsive design
- Persistent cart storage
- Checkout flow

## Tech Stack

- React.js
- Tailwind CSS
- Firebase
- React Router
- Framer Motion
- Context API

## Live Demo

https://zyro-react-firebase.vercel.app

## Installation

```bash
npm install
npm run dev
```
